CalculadoraIMC

Este é um projeto de Calculadora de IMC (Índice de Massa Corporal) desenvolvido em Kotlin para Android Studio.

Funcionalidades
- Cálculo de IMC com base em peso e altura.
- Autenticação de usuário com Firebase.


Configuração Importantes:

Este projeto usa o Firebase para autenticação. Para que o projeto compile e funcione, você precisa seguir estes passos:

1.  Crie o seu próprio projeto no [Firebase Console](https://console.firebase.google.com/).
2.  Habilite a Autenticação (Authentication) e outros serviços que desejar.
3.  Baixe o seu arquivo `google-services.json` pessoal.
4.  Cole este arquivo `google-services.json` dentro da pasta `app/` do projeto.
